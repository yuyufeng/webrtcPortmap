🌐 WebRTC PortMap - HTTP Web Console

通过 WebRTC P2P 连接访问 Agent 本地 HTTP 服务

✨ 功能说明

服务在内网无法访问使用，并不需要公网访问，给自己准备一个随到随用的途径。比如将openclaw的控制端带到任何地方。
方便：轻量级Web穿透，无需安装任何软件即可访问内网服务
将内网的web服务映射到web Console，用于快速验证服务和管理内部网络，不再需要VPN，端口映射
高能：基于webRtc的端口映射，免去访问内网服务要安装VPN的痛苦
下载独立 Client 后，可将 Agent 服务映射到本地端口，按原始 TCP 方式访问，适合数据库、SSH 或本地开发工具直连。
安全：本地密码鉴权，所有数据P2P不通过服务端转发
Agent 本地密码只在 WebRTC DataChannel 建立后用于端到端鉴权，服务端不保存该密码，信令服务只做握手不做转发。
